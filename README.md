## 🏆 The Beginner Checklist

### 🏁 Phase 1: Basic Logic & Control Flow
* [ ] **Console Input/Output:** Learning how to read user input and print results.
* [ ] **Variables & Types:** Understanding strings, integers, decimals, and booleans.
* [ ] **Conditional Logic:** Making decisions using `if/else` statements.
* [ ] **Loops (Iteration):** Making code repeat using `for` and `while` loops.

### 📦 Phase 2: Storing Data & Memory
* [ ] **Collections:** Learning Lists/Dictionaries (Python) and Arrays/Vectors (C++).
* [ ] **C++ Pointers:** Understanding how variables use the computer's physical memory.

### 🏛️ Phase 3: Object-Oriented Programming (OOP)
* [ ] **Classes & Objects:** Learning how to build real-world blueprints for code.
* [ ] **The 4 Pillars:** Understanding Inheritance, Encapsulation, Polymorphism, and Abstraction.

### ⚙️ Phase 4: Beyond the Basics
* [ ] **File Handling:** Reading from and writing data to external `.txt` files.
* [ ] **Error Catching:** Using `try/except` blocks so my programs never crash.

## 💡 My Learning Rules
1. **Side-by-Side:** Learn the logic in Python first, then challenge myself to build it in C++.
2. **No Copy-Paste:** Force myself to type out every single C++ semicolon `;` to build muscle memory.
3. **Keep it Consistent:** Write and push at least a few lines of code regularly.
